# WS EOL mask

Needs to find:

- ` ` (0x20)
- `\r` (0x0D)
- `\n` (0x0A)
- `\t` (0x09)
- `#` (0x23)

Split into sets:

| Code points    | Characters | Desired values |
|----------------|------------|----------------|
| `0x09`,        | `\t`       | 1              |

| `0x20`         | ` `        | 4              |
| `0x23`         | `#`        | 8              |
| Anything else  |            | 0              |
